Comparaison on tooth growth by supp and dose.

Use confidence intervals and/or hypothesis tests to compare tooth growth by supp and dose. (Only use the techniques from class, even if theres other approaches worth considering)

# size of sample dose X supp is 10
n <- 10
#let's compare dose = 2 and supp = VC to dose = 0,5 supp = OJ


g1 <- ToothGrowth[ToothGrowth$supp == "OJ" & ToothGrowth$dose == 0.5,]$len
g2 <- ToothGrowth[ToothGrowth$supp == "VC" & ToothGrowth$dose == 2,]$len
difference <- g2 - g1
mn <- mean(difference); s <- sd(difference); n <- 10
## t confidence interval (95% ... 97,5%-(1-97,5%))
mn + c(-1, 1) * qt(.975, n-1) * s / sqrt(n)
## [1]  7.562025 18.257975
